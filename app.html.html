영어 지문 자동 분석

영어 지문을 붙여넣으면 공우정바른학원 포맷대로 분석교안과 구조도를 자동 생성합니다.
생성한 자료는 자동으로 책장에 저장되어 언제든 다시 열람·인쇄·공유할 수 있습니다.

💡 첫 사용 시 강사 이름을 설정해주세요 (자료 소유자 표시용).

💡 사용 안내
· 분석 생성에는 약 30초~1분이 소요됩니다.
· 생성된 자료는 자동으로 책장에 저장되어 나중에 다시 볼 수 있습니다.
· 책장에서 각 자료를 인쇄 / HTML 다운로드 / 공유 링크 생성 / 재편집 / 삭제할 수 있습니다.
· 다른 강사와 자료를 공유하려면 책장 상단의 📦 백업/복원 기능을 사용하세요.

📌 토큰 발급: 러버블 앱 → 외부연동 → /teacher/integrations 페이지 → "발급" 클릭
💡 사용법: 토큰을 아래에 저장한 후, 책장 카드의 "📚 학습기로 전송" 버튼 클릭

🌐 GitHub Pages 자동 배포 🔴 미설정

💡 분석자료를 GitHub Pages로 자동 배포하여 공개 URL로 학생에게 공유할 수 있습니다.
📌 준비물: GitHub 계정 / 저장소(Public 권장) / Personal Access Token (repo 권한)
📌 GitHub Pages 활성화 필수 (저장소 Settings → Pages → Source: main branch)
📄 GitHub Pages 빠른 시작 가이드 →

👤 기존 책장의 강사명 일괄 변경

💡 책장에 이미 저장된 자료들의 "작성자" 강사명을 한 번에 변경합니다.
📌 예: "Vera"로 작업한 기존 자료를 "원장"으로 일괄 변경
⚠️ 되돌릴 수 없으니 실행 전 📦 백업 권장
